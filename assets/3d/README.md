Coloque aqui o modelo 3D usado na landing page.

Arquivo esperado pelo HTML atual:

```text
assets/3d/modelo.glb
```

Recomendacoes:

- Use `.glb` sempre que possivel, porque ele empacota geometria, materiais e texturas em um unico arquivo.
- Tente manter o arquivo abaixo de 5 MB a 10 MB para a pagina carregar bem.
- Se usar outro nome de arquivo, atualize o atributo `src` do `<model-viewer>` em `index.html`.
